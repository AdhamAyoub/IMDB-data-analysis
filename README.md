# IMDB Movies Data Analysis

## Overview
This project analyzes movie data using Excel. The dataset includes information about movie ratings, genres, actors and finance.

## Files
- `IMBD-movies (original)`: The raw dataset.
- `IMBD-movies (edited)`: The edited dataset.
- `visuals`: containing charts and graphs.
- `README.md`: This file.

## Insights
- Top 10 Popular movies
- Top 10 Directors and their movies salary.
- High budget genres
- Top profited genres
- Durations popularity

## How to Use
1. Open `IMBD-movies (original)` to view the raw data.
2. Open `IMBD-movies (edited)` to view the edited data.
3. Check the `visuals` folder for charts and graphs.
